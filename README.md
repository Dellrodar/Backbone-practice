# Backbone Basics

This repo is based on this udemy course:
["Backbonejs Tutorial"](https://www.udemy.com/course/backbonejs-tutorial/)

This course and repo covers the basics of Backbone:
1. Models
2. Collections
3. Views
4. Events
5. Router
6. Modularizing Backbone
